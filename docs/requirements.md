# 需求分析

## 项目目标

构建一款英语口语陪练工具，让用户在接近真实生活和工作场景的对话中练习英语表达。系统需要完成语音识别、对话推进、纠错反馈、能力评分和课后总结，帮助用户看到自己的表达问题和提升方向。

第一版目标不是覆盖所有英语学习场景，而是把“指定场景下的多轮口语练习”做稳定：用户能选场景、开口说、得到回复、看到纠错和分数，并在结束后看到可执行的总结。

## 目标用户

- 需要提升日常英语表达的学习者。
- 准备英文面试、商务会议或出国旅行场景的用户。
- 希望低成本、高频次获得口语反馈的用户。

## 使用场景

- 点餐：用户在咖啡店或餐厅完成点单、询问推荐、确认价格。
- 面试：用户回答自我介绍、项目经历、职业规划等问题。
- 会议：用户表达观点、确认任务、请求澄清或总结结论。

## 产品边界

### 第一版包含

- Android 端练习主流程。
- 点餐场景作为完整演示场景。
- 面试和会议场景作为可扩展示例。
- 本地规则纠错、基础评分和 fallback 回复。
- FastAPI 后端提供语法检查、教练分析和总结接口。

### 第一版不包含

- 完整账号体系。
- 真实付费、课程体系或学习社区。
- 专业级音素发音评测。
- 大规模场景编辑后台。
- 多端同步。

这些能力不影响当前 Demo 的核心评审路径，后续可在主链路稳定后扩展。

## 核心需求

### 场景选择

- 用户可以选择预置练习场景。
- 每个场景由 JSON 脚本描述，包括角色、目标、关键词、推荐表达和多轮对话规则。
- 后续可以扩展自定义场景。
- 进入场景前展示场景名称、目标、角色和难度。

### 语音输入

- 支持按住说话和点击说话。
- 支持实时 partial text 展示。
- 支持长句作为一个 Turn 处理。
- 单轮建议最长 30 秒，超时后提示用户提交或继续。
- 识别失败时允许用户重试，不直接结束练习。

### 对话推进

- 系统根据当前场景、用户表达和脚本状态生成下一轮回复。
- Demo 阶段优先使用规则和脚本推进，增强阶段接入 LLM。
- 网络或后端不可用时使用 Demo fallback 数据。
- 每轮对话推进后记录本轮文本、纠错结果、评分、AI 回复和状态来源。

### 纠错反馈

- 展示用户原句和推荐表达。
- 标出常见语法、用词和礼貌表达问题。
- 本地规则作为主链路，LanguageTool 作为增强能力。
- 纠错提示在本轮提交后展示，避免在用户说话中频繁打断。

### 能力评分

评分维度包括：

- 语法准确度
- 流利度
- 发音清晰度
- 场景完成度

评分需要可解释，Demo 阶段采用规则公式，避免黑盒评分。

每个维度都需要显示简短原因，例如“缺少礼貌表达”“ASR 置信度较低”“已完成点餐目标”。

### 课后总结

- 汇总本次练习的轮次、平均分和关键问题。
- 展示推荐表达和下一次练习目标。
- 支持查看历史练习记录。

### 视觉体验

- 练习页采用横屏舞台。
- 背景图用于表现当前场景。
- 2D 角色根据状态切换 idle、listening、thinking、speaking 动画。

## 非功能需求

- 响应时间：主链路反馈尽量控制在 3 秒内。
- 可用性：无网络或后端异常时仍能完成 Demo 流程。
- 兼容性：优先支持 Android 8.0 及以上。
- 隐私：语音数据只在明确需要时发送后端，不在仓库中保存真实用户录音。
- 可维护性：核心逻辑模块化，场景脚本和评分规则可独立维护。
- 可测试性：纠错、评分、场景推进等核心逻辑需要有自动化测试。
- 可演示性：任何增强模块失败时，Demo 都能展示完整闭环。

## 验收标准

### 练习主链路

- 用户可以进入点餐场景。
- 用户完成至少 3 轮语音或文本输入。
- 每轮展示用户原句、推荐表达、AI 回复和四维评分。
- 结束练习后展示课后总结。

### 反馈质量

- 常见表达错误可以被纠正，例如 `I want order a coffee`。
- 礼貌表达缺失可以被提示。
- 评分结果能解释分数来源。
- 场景完成度随目标完成情况变化。

### 稳定性

- 后端不可用时仍返回 fallback 回复和本地评分。
- 单轮识别失败后可以重试。
- 场景脚本缺字段时能给出明确错误。

## Demo 优先级

### P0

- 场景选择
- 横屏练习页
- Android SpeechRecognizer 识别
- Android TextToSpeech 播放
- 本地规则纠错
- 可解释评分
- Demo fallback
- 点餐场景完整闭环

### P1

- LanguageTool 增强纠错
- 课后总结
- 角色状态动画
- 历史记录
- FastAPI 基础接口

### P2

- PaddleSpeech 备用 ASR
- LLM 个性化表达建议
- 文生图场景背景缓存
- 多场景扩展编辑器
