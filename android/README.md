# Android 客户端

客户端计划使用 Kotlin + Jetpack Compose 实现。

## 主要模块

- 场景选择页
- 横屏练习页
- 课后总结页
- `SpeechRecognizerAdapter`
- `TextToSpeechAdapter`
- `TurnBuffer`
- `ScenarioEngine`
- `RuleCorrectionEngine`
- `ScoreEngine`
- `DemoFallbackRepository`

## 实现顺序

1. 建立最小 Android 工程。
2. 实现场景选择和横屏练习页。
3. 接入 SpeechRecognizer 与 TextToSpeech。
4. 实现本地脚本对话、纠错和评分。
5. 接入后端增强能力。
